Aionic

Aionic offers open source applications for project management and collaboration. Our focus is on simplifying and accelerating the workflow for agile teams in the open source way.

Core

Aionic Core is the foundation for any other of our services. It's the place to store and manage all the data you create.

  • REST API
  • MySQL ORM
  • GitHub API Integration
  • Permission control via ACL
Get started GitHub

Milestone

Aionic Milestone is a service for project management. You can create tasks and assign them to your team members.

  • Projects
  • Kaban Boards
  • Login / Registration
  • User / Task Management
